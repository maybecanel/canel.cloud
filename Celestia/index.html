Application Crash Detected

To resolve the issue, follow these steps:

  1. Press Win + R on your keyboard to open the Run dialog box.
  2. Type %localappdata% and press Enter.
  3. In the window that appears, find the folder named "FortniteGame."
  4. Once you locate the "FortniteGame" folder, right-click on it and select "Delete."

This should help resolve the problem.

Easy Installer Closes

First, install .NET and try again. If this does not resolve the issue, download and install the 1.1.1.1 VPN and attempt once more. These steps should fix the problem unless b1k is down.

Reasons why your build might not be launching:

  • The build is installed in your personal OneDrive folder: Our launcher does not have access to files within OneDrive. Please copy your build to a different location and ensure the path does not include OneDrive
